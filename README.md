# GravityPlatform2D

## Overview
**GravityPlatform2D** is a small 2D game prototype where the player moves on a static rectangular platform and is constantly "pulled" to the surface by gravity. The player can move, jump, and smoothly navigate corners while never falling off the platform.

## Game Mechanics

- **Movement**: The player walks left and right along the current edge of the platform.
- **Jumping**: The player jumps "up" relative to the current surface.
- **Corner transitions**: The player can smoothly move around the corners of the platform.
- **Always grounded**: The player is always attached to the "bottom" of the current edge.

## Controls

### Keyboard (PC)
- **A / D** — move left/right
- **Space** — jump

### Touch (Mobile)
- **UI buttons**:
  - Left / Right move buttons
  - Jump button

## Requirements
- Unity 6000.2.10f or later
- 2D Physics enabled
